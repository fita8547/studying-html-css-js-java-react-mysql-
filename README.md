# 풀스택 개발 학습 저장소

## 📚 학습 강좌
**고품질 리액트와 스프링부트 65시간 풀스택 개발 강의(도커, AWS, GITHUB-ACTIONS)**

> 1인 IT 창업, 1인 사업가 모두 경험한 저자가 만든 솔직하고 담백하며 직관적이고 깊은 리액트+스프링부트 강좌 (도커, AWS, GITHUB-ACTIONS)

## 🎯 학습 목표
- React 기초부터 고급 개념까지 완전 정복
- Spring Boot를 활용한 백엔드 개발
- Docker를 이용한 컨테이너화
- AWS 클라우드 배포
- GitHub Actions를 통한 CI/CD 구축
